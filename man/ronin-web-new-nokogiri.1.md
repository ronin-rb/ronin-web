# ronin-web-new-nokogiri 1 "May 2022" Ronin "User Manuals"

## SYNOPSIS

`ronin-web new nokogiri` [*options*] [*FILE*]

## DESCRIPTION

Generates a new nokogiri Ruby script for parsing HTML/XML.

## ARGUMENTS

*FILE*
  The file to create.

## OPTIONS

`-U`, `--url` *URL*
  Optional URL for the script.

`-h`, `--help`
  Print help information
